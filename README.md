# Figma to HTML Product Page (Swiper + Dynamic Loading)

This project converts a Figma design into a fully responsive product listing web page using **HTML, CSS, and Vanilla JavaScript**, with advanced UI features like Swiper slider, lazy loading, dynamic product rendering, and popup previews.

---

## 🚀 Features

### 🖼 Hero Section
- Full-width static banner
- Responsive image background
- Optimized for mobile and desktop

### 🛍 Featured Products (Swiper Carousel)
- Built using **Swiper.js**
- Auto sliding + navigation arrows
- Responsive touch support

### 📦 Product Listing Section
- Products loaded dynamically from a **third-party API / library**
- Initial render: **14 products**
- Dropdown allows adding more products 

### ⚡ Lazy Loading
- Product images load on scroll
- Improves performance and reduces initial load time

### 🔽 Dropdown Feature
- Select number of products to append
- Dynamically updates product grid without refresh

### 🖼 Product Popup Modal
- Click product image → opens modal
- Shows:
  - Image
  - Product ID
  - Close button

### 📱 Responsive Navigation
- Sticky navbar
- Sections:
  - Home
  - Featured Products
  - Product Listing
- Active link highlighting on scroll
- Mobile hamburger menu support

---

## ⚡ How to Run

1. Clone the repository   
2. Open `index.html` in your browser  

## 🧰 Tech Stack

- HTML5
- CSS3 (Flexbox + Grid)
- JavaScript (ES6+)
- Swiper.js
- Intersection Observer API (Lazy Load)
